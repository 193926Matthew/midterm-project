Forest Fire Animation Project

Code

crickit.servo1.setAngle(180) crickit.servo2.setAngle(0) forever(function () { if (crickit.touch1.touchRead() >= 800) { pause(2000) crickit.servo1.setAngle(90) pause(100) pause(1000) crickit.servo2.setAngle(180) pause(30000) crickit.servo1.setAngle(180) pause(1000) crickit.servo2.setAngle(0) } }) forever(function () { light.graph(crickit.touch1.touchRead(), 1000) })

This code is used to both move the bucket of water to spill its water on the fire and move the fire downwards when it is extingusihed. This code works by utilizing a touch sensor to activate cricket motors when a person touches it. The cricket motors, once activated, will change their rotation and since the motors are connected to both the ffire and bucket they'll rotate the props accordingly.
